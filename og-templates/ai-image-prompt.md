```
Create a professional social media Open Graph image (1200x630px) for [PRODUCT NAME] with the following specifications:

**Visual Style:**
- Modern, premium tech company aesthetic
- Clean, minimalist design with subtle gradients
- Professional B2B software branding
- High-quality, crisp graphics suitable for social media sharing

**Color Palette:**
- Primary background: Deep navy blue (#0B111D) to medium blue (#2D4263)

- Gradient: Smooth diagonal gradient from dark navy to teal

**Typography:**
- Font: Plus Jakarta Sans or similar modern sans-serif
- Main title: Bold, large (70-80px), white text
- Subtitle: Medium weight (24-28px), slightly transparent white
- Product badge: Small caps, teal background, rounded corners

**Layout Structure:**
- Left side: Text content (brand name, product title, description, key features)
- Right side: Professional icon/visual element in a glass-morphism card
- Bottom right: Small domain text "konnektr.io"
- Top right: Pricing or "Coming Soon" badge

**Icon/Visual Requirements:**
- [SPECIFIC ICON REQUIREMENTS - see individual products below]
- Placed in a frosted glass card with subtle backdrop blur
- Soft drop shadow and subtle glow effect
- Professional, vector-style appearance
- Avoid cartoon-like or amateur aesthetics

**Background Elements:**
- Subtle geometric patterns or dots in very low opacity
- Professional tech/data visualization hints
- No distracting elements that compete with text
- Clean, spacious feel with proper breathing room

**Quality Standards:**
- High resolution, crisp edges
- Professional color harmony
- Excellent contrast for text readability
- Social media optimized composition
- Corporate/enterprise software aesthetic
```

## Additional Instructions for AI Generator

**Do:**

- Prioritize professional, enterprise software aesthetics
- Use subtle shadows and gradients for depth
- Ensure excellent text contrast and readability
- Create icons that look like they belong in a premium SaaS product
- Maintain consistent brand colors across all variations
- Include subtle background patterns that enhance without distracting

**Don't:**

- Use cartoonish or amateur-looking icons
- Make backgrounds too busy or distracting
- Use colors outside the specified brand palette
- Create low-contrast text that's hard to read
- Include too many visual elements that compete for attention
- Use generic stock photo elements

**Style References:**
Think "Stripe", "Vercel", "Linear", or "Notion" - clean, modern, professional B2B software branding with excellent typography and sophisticated color use.
